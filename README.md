โครงการ " Cafeteria Seats Tracker ระบบตรวจสอบและติดตามจำนวนที่นั่งในโรงอาหารแบบอัจฉริยะ"
โครงงานนี้ช่วยเพิ่มความสะดวกสบายให้แก่นักศึกษา ลดระยะเวลาในการหาที่นั่ง 
และสนับสนุนการจัดการทรัพยากรภายในโรงอาหารให้มีประสิทธิภาพมากยิ่งขึ้น 
อีกทั้งยังสามารถนำข้อมูลที่เก็บได้ไปใช้ในการวางแผนปรับปรุงหรือขยายพื้นที่โรงอาหารในอนาคต 
รวมถึงช่วยลดความแออัดซึ่งส่งผลดีต่อด้านสาธารณสุขภายในคณะ
